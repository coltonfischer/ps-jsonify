### [](#header-3)Description

Use this property to return the length of the JSON Array.

### [](#header-3)Returns

Integer.

### [](#header-3)Example

```java
Local PSM_JSON:Array &oArray = create PSM_JSON:Array();

Local integer &iValue = &oArray.Length;
```
