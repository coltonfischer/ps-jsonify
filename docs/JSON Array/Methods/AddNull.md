### [](#header-3)Syntax

**AddNull**()

### [](#header-3)Description

Use this method to add a Null value to the JSON Array.

### [](#header-3)Parameters

None.

### [](#header-3)Returns

None.

### [](#header-3)Example

```java
Local PSM_JSON:Array &oArray = create PSM_JSON:Array();
   
&oArray.AddNull();
```
