### [](#header-3)Description

Use this property to return the number of properties of the JSON Object.

### [](#header-3)Returns

Integer.

### [](#header-3)Example

```java
Local PSM_JSON:Object &oObject = create PSM_JSON:Object();

Local integer &iValue = &oObject.Size;
```
